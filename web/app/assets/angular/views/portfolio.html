M-InfoMas

M~InfoMaS is a management system which can be used by Maharishi Arvind Institute Of Engineering & Technology, Jaipur to store, manage and process data regarding there day to day work and can use that data to draw helpful information for decision making.

This system provides a very effective way to record, process and retrieve information. In this system we can store many kind of information. In this we have provided different forms for each type of information. Every form can be used to fill, change or retrieve some kind on information in its own way. Every forum also has their level of authentication required so only a user with at least that much authentication can access that forum.
